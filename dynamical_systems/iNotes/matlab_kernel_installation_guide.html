1. Overview¶

  • 내용
    • 본 매뉴얼은 Jupyter notebook의 Matlab 을 위한 패키지 설치를 다룸
  • OS
    • 사용자 편의를 위해 Windows 10으로 매뉴얼 작성
  • Anaconda
    • 본 메뉴얼에서는 패키지들 사이의 충돌을 방지하기 위해 Anaconda를 사용하지 않음

2. Requirements¶

  • Windows 10, 64 bits PC (recommended)
  • Python
  • Visual C++ 2015 Redistributable (or 2017)
  • Matlab

3. Matlab Kernel Installation¶

  • Python용 Matlab Engine API 설치하기

    [1] cmd 관리자모드 실행

[2] Matlab이 설치된 경로/extern/engines/python로 경로 변경

[3] python setup.py install 입력

  • Matlab Kernel 설치하기

    [4] pip install matlab_kernel 입력
