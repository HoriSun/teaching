1) Write a python code to conduct a linear regression (least square).

In [6]:
x_reg = x_reg.reshape(-1, 1)
y_reg = y_reg.reshape(-1, 1)
ones_col = np.ones((x_reg.size, 1))
phi = np.hstack([x_reg, ones_col])

w = cvx.Variable((2,1))
objective = cvx.Minimize(cvx.sum_squares(phi * w - y_reg))
prob = cvx.Problem(objective)
prob.solve()
w = w.value

plt.figure(figsize = (10, 8))
plt.plot(x_reg, y_reg, 'k.', label = 'data')
plt.plot(x_reg, x_reg * w[0] + np.ones((200, 1)) * w[1], color = 'red', label = 'linear regression', linewidth = 4.0)
plt.xlim(-4,4)
plt.ylim(-4,4)
plt.legend()
plt.show()

2) Write a python code to perform PCA. Since mean(x) = mean(y) = 0, normalization step (mean subtraction and rescaling) can be skipped.

In [7]:
num_data = x_reg.size
data = np.hstack([x_reg, y_reg])
data = np.asmatrix(data)
S = 1 / (num_data - 1) * data.T * data

D, V = np.linalg.eig(S)
idx = np.argsort(-D)
D = D[idx]; V = V[:, idx]
h = V[1, 0] / V[0, 0]
xp = np.arange(-4, 4, 0.1)
yp = h * xp;

plt.figure(figsize = (10, 8))
plt.plot(x_reg, y_reg, 'k.')
plt.plot(xp, yp, 'r', linewidth = 4.0)
plt.xlim(-4,4)
plt.ylim(-4,4)
plt.show()

Problem 3¶

Describe the difference between the process of finding the linear regression line and the PCA line.

Figure 6

Solution 3¶

The process of minimizing error is different.

  • Linear regression error: minimization of error between real values and prediction values

  • PCA error: maximize data variance (minimization of error between projected data in line and training data)
